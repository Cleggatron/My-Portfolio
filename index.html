Clear Sky Group Project

Link To Repo

Technologies Used: HTML, CSS, JS, Semantic UI

This website accepts a user input for location, and returns the weather for that location, as well as the major celestial bodies that would be visible and the location for them. This allows the user assess whether it is worth going out to look at the night's sky. This was group project aimed at harvesting data from multiple API's utilising user input.

Workday Planner

Link To Repo

Technologies Used: HTML, CSS, JS, JQuery, Bootstrap, Moment.js

This website allows a user to plan their day's tasks in hourly intervals. The schedule saves to local storage, meaning the tasks will not be lost to a page refresh/accidental closure. Additionally the intervals are colour coded to display, past, present, and future intervals. There is an hourly refresh to update this coding.
